# wezterm_config

nvim の操作感に統一した WezTerm 設定。

## LEADER キー

`wezterm.lua` で定義（デフォルト: `Ctrl+Space` など）。

---

## メインキーバインド

### ペイン操作

| キー | 動作 |
|------|------|
| `LEADER + h/j/k/l` | 左/下/上/右のペインへ移動 |
| `LEADER + z` | ペインのズーム切り替え |
| `LEADER + x` | 現在のペインを閉じる |

### タブ操作

| キー | 動作 |
|------|------|
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 次/前のタブへ |
| `Super+T` | 新しいタブを開く |
| `Super+W` | 現在のタブを閉じる |
| `LEADER + {` / `}` | タブを左/右に移動 |

### その他

| キー | 動作 |
|------|------|
| `Shift+Meta+F` | フルスクリーン切り替え |
| `LEADER + Space` | ワークスペース一覧 |
| `Super+P` / `Ctrl+Shift+P` | コマンドパレット |

## モード

右ステータスバーに現在のモードが色付きで表示される（NORMAL=緑、RESIZE=黄、WINDOW=青）。

### コピーモード（`LEADER + [`）

nvim の Normal/Visual モード相当。

**移動**

| キー | 動作 |
|------|------|
| `h/j/k/l` | 左/下/上/右 |
| `0` / `$` / `^` | 行頭/行末/非空白行頭 |
| `w` / `e` / `b` | 単語単位で前進/後退 |
| `g` / `G` | スクロールバック先頭/末尾 |
| `H` / `M` / `L` | ビューポートの上/中/下 |
| `Ctrl+F` / `Ctrl+B` | ページ下/上 |
| `Ctrl+D` / `Ctrl+U` | 半ページ下/上 |

**選択・ヤンク**

| キー | 動作 |
|------|------|
| `v` / `V` / `Ctrl+V` | 文字/行/矩形選択 |
| `o` / `O` | 選択範囲の反対端へ |
| `y` | コピーしてモード終了 |

**検索**

| キー | 動作 |
|------|------|
| `/` | 前方検索 |
| `n` / `N` | 次/前のマッチへ |
| `Ctrl+R` | マッチタイプ切り替え |
| `Ctrl+U` | 検索パターンをクリア |

**終了**: `q` / `Escape` / `Ctrl+[`

---

### ウィンドウモード（`LEADER + w`）

nvim の `<C-w>` 相当。ワンショット（1操作で自動終了）。

| キー | 動作 |
|------|------|
| `h/j/k/l` | ペイン移動 |
| `w` / `p` | 次/前のペインへ |
| `s` / `v` | 水平/垂直分割 |
| `z` | ズーム切り替え |
| `c` | ペインを閉じる |
| `n` | 新しいタブ |
| `r` | リサイズモードへ |
| `Escape` / `q` | モード終了 |

---

### リサイズモード（ウィンドウモード → `r`）

ペインサイズを連続調整できる（`q` で抜けるまで維持）。

| キー | 動作 |
|------|------|
| `h/j/k/l` | ペインを左/下/上/右に 3 単位拡縮 |
| `Escape` / `Enter` / `q` | モード終了 |
